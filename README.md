## Getting started

To get started with this project:

```bash
  git clone https://github.com/monlasan/Hireleak-ATS.git
```

and copy the .env.sample variables into a separate .env file, fill them out and following run the commands:

```bash
  yarn install
  yarn dev
  yarn build
```
